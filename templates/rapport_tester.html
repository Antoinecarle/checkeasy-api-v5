Comment ameliorer les prompts IA ?

Cet outil vous permet de tester vos modifications de prompts sur des rapports reels, sans impacter les donnees existantes.

1

Identifier un probleme

Selectionnez un rapport dans la liste a gauche. Parcourez les pieces et comparez les photos avec l'analyse IA. Reperer les scores incorrects, problemes manques ou faux positifs.

2

Modifier les prompts

Allez dans l'admin des prompts pour ajuster les instructions donnees a l'IA. Vous pouvez modifier le ton, les criteres de notation, les categories de problemes, etc.

Ouvrir Prompts Admin →
3

Re-analyser une piece

Revenez ici, cliquez sur "Re-analyser cette piece" sur la piece concernee. L'API relance l'analyse avec vos nouveaux prompts (sans modifier le rapport original).

4

Comparer ancien vs nouveau

Le resultat s'affiche en comparaison cote-a-cote : ancien score/problemes vs nouveau. Iterez jusqu'a obtenir le comportement souhaite, puis deployez les prompts.
