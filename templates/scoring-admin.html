Pénalités par Sévérité

Points retirés de la note (5) pour chaque issue selon sa sévérité.
Exemple: 1 issue HIGH (-1.0) + 2 issues MEDIUM (-0.4×2) = Note finale: 5 - 1.0 - 0.8 = 3.2/5
📂

Multiplicateurs par Catégorie

Multiplicateur appliqué à la pénalité selon le type d'issue.
Exemple Ménage: Issue propreté (×1.5) = pénalité plus forte. Issue positionnement (×0.8) = moins grave.
🏠

Poids d'Importance par Pièce

Poids utilisé pour la moyenne pondérée des notes.
Exemple: Cuisine (×2.0) compte 2× plus que Chambre (×1.0) dans la note globale.
🎲

Seuil de Confiance

Confiance minimum pour qu'une issue soit comptée dans le score.

Labels par Plage de Notes
